# Start Here

Welcome to the AI Development Playbook. This is the quickest, safest way to get started with AI-assisted development.

## Where to Start

**Brand new to AI-assisted development?**
→ Read the [Full SOP](sop.html) first to understand the complete process, then use [Quick Reference](quick-reference.html) as your daily companion

**Have some experience with AI coding tools?**
→ Jump straight to [Quick Reference](quick-reference.html), consult the Full SOP for edge cases

**Setting up a new project?**
→ Start with [AI Tools & Setup](?doc=content/shared/ai-tools-and-setup.md), then create your [Constitution](?doc=content/shared/example-constitution.md)

**Just need to look something up?**
→ Browse the [Library](#library-shared-resources) below or use search

## Principles to keep in mind
- Follow phase gates; don't skip checkpoints.
- TDD-first with coverage thresholds; raise blockers early.
- Reviewer independence: no author-self-review shortcuts.
