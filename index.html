Getting Started

Your journey to high-quality AI-assisted development starts here

1

Understand the Why

AI tools generate code incredibly fast, but without process, teams ship bugs, drift from architecture, and waste time on rework. This framework ensures AI builds exactly what you need, with quality baked in.

2

Choose Your Path

First time? Read the full SOP to understand the complete workflow.

Need a refresher? Jump to the Quick Reference for daily guidance.

Leading a team? Review the Customization Guide in the SOP.

3

Customize for Your Org

Before distribution, update document owner, adjust paths for your project structure, configure tool setups for your tech stack, and adapt story ID formats to your conventions.

4

Start with Phase 0

Every feature begins with foundation work: understand requirements, read your constitution, create/review specifications, and perform pre-implementation analysis. Invest 30-60 minutes upfront to save hours of rework.

5

Follow the 5-Phase Flow

Foundation ‚Üí Specification ‚Üí GitHub Issues ‚Üí Implementation (TDD) ‚Üí Multi-Model Review. Each phase has clear checkpoints. Don't skip gates‚Äîthey catch issues early when they're cheap to fix.

6

Master Test-First

TDD is non-negotiable with AI code. Write tests FIRST (RED), implement to pass (GREEN), then refactor (CLEAN). AI-generated code looks correct but has hidden edge cases‚Äîtests catch them immediately.

Key Principles

What makes AI-assisted development different

Specs Are Mandatory

A clear 30-minute spec saves hours of AI going in circles. Quality of spec = quality of output. Without detailed specs, AI builds the wrong thing efficiently.

Test-First Always

AI code looks correct but has hidden bugs. Writing tests first (TDD) catches edge cases immediately. 80% coverage minimum, no exceptions.

Multiple AI Reviews

Same AI makes same mistakes. Code by Claude ‚Üí Review by Copilot catches 40% more bugs. Different models have different blind spots.

üìö

Constitution Enforces Patterns

AI can memorize and enforce architectural patterns better than humans. Your constitution ensures consistency across all features and developers.

Invest Upfront
